## Setup (Terminal Based)
 Dependencies 
    Python<br>
        - pandas<br>
        - matplotlib<br>
        - streamlit

To start run:
```bash
python main.py
```
<br>
The streamlit version is stored in web.py and can be runned by:

```bash
streamlit run web.py
```
